# Mellringe Café Website

Static informational website.

No cart, no checkout, no payment, no ordering system.

## Files
- index.html
- style.css
- script.js
- images/

## How to use
Open `index.html` in your browser.

Replace `#` links for Instagram and TikTok with the real links later.

## Latest update
Added Meze & sallader category: Tabbouleh, Fattoush, Bulgur, Hummus, Mutabbal with images.


## Admin page

Open `admin.html`.

Demo login:
- Email: `admin@mellringecafe.se`
- Username: `admin`
- Password: `Mellringe2025!`

Features:
- Add/edit/delete menu categories
- Add/edit/delete food items and prices
- Change category image path
- Edit gallery images
- Edit café phone, address, social handles, hero text and about text
- Export data as JSON

Important: this is a static frontend demo using browser localStorage. For real production security, connect it to a backend with database, hashed passwords, server sessions/cookies, and proper image upload storage.
